Proyecto desarrolado para la materia de Desarrollo de aplicaciones moviles usando el framework de Ionic para la creacion del mismo.
El proyecto cuenta con las siguientes funcionalidades:
-Splash screen
-Conexion con Firebase para Autenticacion y notas de peliculas.
-Perfil de usuario.
-Constante muestra del estado del sistema mediante pop ups.
-Listado de peliculas usando el API de omdbAPI.
-Detalle de una pelicula.
-CRUD de notas sobbre peliculas almacenados en Firebase.
-Geolocalizacion usando los plugins de Cordova y mostrando a traves de Google maps.
-Informacion de cantidad de Usuarios acceden a la aplicacion con Google Analitycs.
-Responsive Design.
-Compartir peliculas por Twitter.
